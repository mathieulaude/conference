🔹 GitOps : simplifier les déploiements grâce à Git

Déployer une application ne devrait pas être une opération risquée ou opaque.
Pourtant, dans de nombreuses équipes, la mise en production reste complexe, peu traçable et difficile à maîtriser.

Le GitOps propose une approche simple et moderne :
faire de Git la source unique de vérité pour piloter les déploiements et la configuration des environnements.

Lors de cette session accessible aux débutants, nous verrons :
• ce qu’est réellement le GitOps, sans jargon technique
• les problèmes concrets qu’il permet de résoudre
• les principes clés pour bien démarrer
• les bonnes pratiques essentielles pour travailler en équipe

La présentation se conclura par une démonstration guidée, montrant comment un simple changement dans Git peut déclencher automatiquement un déploiement, avec traçabilité et retour arrière immédiat.

👉 Aucune connaissance de Kubernetes n’est requise.
Une session idéale pour découvrir le GitOps et repartir avec une vision claire, concrète et directement exploitable.
